The majority of blobs that are used in this project are not self-made. This page provides you a list of sources for all blobs that we use.
				Please note that we do not know all actual sources.
				We try our best to give credits to all blob creators, so if you own one of these and are no listed as the creator,
				feel free to open an issue on the Github repository.
				If you feel like you do not want us to use your creation, also feel free to open an issue. We will remove it as fast as possible.
				Do note that you need to prove that you really are the creator.
